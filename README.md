# Cyber_Security_Awareness_Chatbot_Part2
Lumina Cybersecurity Chatbot
Lumina is a WPF cybersecurity awareness chatbot developed with C#
The chatbot helps users learn about cybersecurity topics such as phishing, passwords, malicious software,firewalls and overallonline safety.

Features
.GUI built with WPF
.Voice greeting
.Keyword recognition
.Random responses
.Sentiment detection
.Memory recall
.Conversation flow
.Delegate implementation
.File handling
.User friendly interface

Technologies used
.C#
.WPF
. .NET Framework 4.8
.Visual Studio 2022

Example Topics
.Password safety
.Phishing
.Malicious software
.Firewalls
.Cybersecurity awareness
